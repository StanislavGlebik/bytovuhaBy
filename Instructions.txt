Версия django - 1.5.4
Версия python - 2.7.5

В репозитории кроме кода находится также и пример простой базы данных (bytovuha.db).

Запуск сервера - python manage.py runserver
По такому url - http://127.0.0.1:8000/ - будет стартовая (пока пустая) страница. 

По такому - http://127.0.0.1:8000/admin/ - находится админка. Username - stash93, пароль - 111111.

Такими урлами можно делать запросы:
	 http://127.0.0.1:8000/user/d+/name - где /d+ - id юзера. В ответ возвращается имя юзера.
	 http://127.0.0.1:8000/product/d+/ - где /d+ - id товара. В ответ возвращается описание товара.
