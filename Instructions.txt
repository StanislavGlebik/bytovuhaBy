Версия django - 1.5.4
Версия python - 2.7.5

1. Запуск сервера
	
	Запуск сервера - "python manage.py runserver".
	Или лучше "python manage.py runserver 0.0.0.0:8000" - тогда сервер будет виден и из внешней сети.
	Можно еще воспользоваться файликом "runserver.bat" из корня проекта.

	По такому url - http://127.0.0.1:8000/ - стартовая страница. 

2. База данных	
	
	Перед тем, как работать нормально, нужно создать базу данных. Для этого нужно
		
		1. Остановить сервер (закрыть окно консоли, в которой он работает)
		2. Удалить файлик bytovuha.db в корне проекта (файлика может и не быть)
		3. Запустить python manage.py syncdb
		4. Запустить set DJANGO_SETTINGS_MODULE=bytovuhaBy.settings
		5. Запустить python filldb.py
		6. Запустить сервер
		Не должно быть никаких ошибок, если все сделано правильно
		Шаги со второго по пятый делает скрипт "syncdb.bat", который находится в корне проекта, можно пользоваться им

	По такому адресу - http://127.0.0.1:8000/admin/ - находится админка. При создании базы будет предложено также создать админку.
